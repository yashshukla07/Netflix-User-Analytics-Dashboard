# Netflix-User-Analytics-Dashboard

**Project Overview**

This project presents a two-page interactive Netflix analytics dashboard designed to analyze user activity, subscription behavior, content distribution, and viewing patterns.
The dashboard helps stakeholders understand who is watching, what they are watching, how long they watch, and how content trends evolve over time.

**Business Problem**

- Streaming platforms need data-driven insights to:
- Understand user engagement and inactivity
- Analyze subscription-wise watch behavior
- Identify popular genres, countries, and content types
- Track content growth and release trends over time

This dashboard converts raw Netflix data into actionable insights for content strategy, user retention, and platform optimization.

**Tools & Technologies**

- Power BI – Interactive dashboards & visuals
- Excel / CSV Dataset – Netflix user & content data
- DAX – Measures, KPIs, averages, and aggregations
- Data Modeling – User, content, genre, country, and date relationships

**Key Business Insights**

- Premium users show higher average watch time, indicating stronger engagement
- Certain genres dominate across age groups, reflecting audience preferences
- Content production is concentrated in a few countries, led by the US and India
- Netflix content growth peaked during specific years, showing platform expansion phases
- Monthly release patterns highlight seasonal content strategies

**Questions This Dashboard Answers**

- Which user segments are most active on Netflix?
- How does watch time vary by age group and subscription type?
- What genres and content types dominate the platform?
- Which countries contribute the most content?
- How has Netflix’s content library evolved over time?
